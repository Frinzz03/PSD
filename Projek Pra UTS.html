Analisa Data Ecoli Menggunakan Tool Knime#

Tools yang di butuhkan :

  • Download dan install Knime https://www.knime.com/downloads

  • Download Driver PostgreSQL di https://jdbc.postgresql.org/download/

  • Download dan install Miniforge di https://conda-forge.org/miniforge/#history

Konfigurasi Knime#

Membuat workflow baru#

Setelah Kniime berhasil di install lakukan tahapan ini

  • Masuk ke Local Sapce pada panel kiri dari beranda KINME

  • Buat workflow baru dengan menekan tombol “+” pada bagian kanan

  • Masukkan nama workflow kemudian tekan create

Menginstall extention#

Pada bagian kanan masuk ke menu -> Install Extention

Setelah itu cari extention yang akan di install

Konfigurasi PostgreSQL#

  • Masuk ke Preference pada bagian kanan atas

  • Pilih KNIME -> Database, kemudian klik Add untuk menambahkan driver dari database PostgreSQL

  • Isi ID, Name, Description

  • Klik Add file dan pilih driver PostgreSQL yang telah di download

  • Klik Fide driver classes di bagain baawah kemudian klik OK

Setelah database telah ditambah klik Apply and close

Proses analisa menggunakan KNIME#

Mengambil data dari PostgreSQL#

  • Pada node PostgreSQL Connector dan lakukan konfigurasi. Setelah itu klik Apply dan OK kemudian execute nodenya
